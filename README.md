# farewell for Bea

good-bye gift for Bea, from the team.


## LEDs

WS2812B programmable LED strips.
60 LEDs per meter.

power consumption per 10 LEDs:
- `0.073A` - black (2m - idle)
- `0.188A` - red
- `0.131A` - green
- `0.188A` - blue
- `0.416A` - white
